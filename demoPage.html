That's why there's a third option. Luckily, you can create a simple true|false|null toggler for all those times when choosing NOT to decide is still a choice.

Using some css, you can style a range input element to resemble a toggle slider. By setting the min|max parameters to 0 and 2, then setting the step parameter to 1, the range input will snap only to the values zero, one, and two. Using css variables allows the colors to change.

The tfx script will loop through and do that to all input elements with the class "tfx tfx-slider" 


<input
 type="range"
 class="tfx tfx-slider"
/>

The script allows several options, customizable through the data-options paramater, aswell as with styles. 


<input
 type="range"
 class="tfx tfx-slider"
 style="
    width: 250px;
    height: 75px;
 "
 data-options='{
    "value" : 2,
    "trueColor" : "#080",
    "falseColor" : "var(--samp-var)",
    "disabledColor" : "rgba(200,0,0,0.1)",
    "transSpeed" : "0.3s",
    "step" : 1,
    "disabled" : false
 }'
>

Watch that data-options value; it has to be a well-formated JSON string.

value, disabled, transSpeed, and colors can all be set from methods:

.setValue(int);

.setDisabled(boolean);

.setTrans(css-trans-string);

.setColor(nameofColor, valid-css-color-value);

Each of those dispatches an event. The detail of the valueSet event is the string of the value 0, 1, or 2 - so that has to be parsed, or converted, if needed. Every thing else dispatches whatever val got set.

The ynxScript.js creates option labels on either side of a true|false|null toggler, by converting every div with the class "ynx ynx-holder" and styling them according to the same data-options parameter from the t|f|x script. The ynxScript.js was used for all the labelled toggles on this page.

After creating an instance of the ynx-holder, listen to its valueSet event, and respond accordingly. This code creates the example that follows. 


<div
  id="question05"
  class="ynx ynx-holder"
  data-caps='["up","dn"]'
  data-options='[
      "returnValues" : [
          "upward",
          "not selected",
          "downward"
      ]
  ]'
></div>

<div>
  database entry value:
  <i id="dbEntry05">
    not selected
  </i>
</div>

<script>
     
  document
  .getElementById('question05')
  .addEventListener('valueSet', e => {
      document
      .getElementById('dbEntry05')
      .innerText = e.detail
  });
    
</script>
